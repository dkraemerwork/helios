## Steps

### Step 0 ‚Äî Pre-flight checks

- Verify `/Users/zenystx/IdeaProjects/helios/` exists and is empty
- Verify no uncommitted changes in `helios-1/`
- Run `bun test` in `ts-port/` ‚Üí capture baseline (expect 2105 pass)
- Run `bun test` in `ts-port-app/` ‚Üí capture baseline (expect 25 pass)

### Step 1 ‚Äî Copy ts-port/ ‚Üí repo root

```bash
rsync -av --exclude='node_modules' --exclude='dist' \
  helios-1/ts-port/ helios/
```

Copies: `package.json`, `tsconfig.json`, `tsconfig.build.json`, `bunfig.toml`, `bun.lock`, `helios-server.ts`, `README.md`, `src/`, `test/`, `scripts/`, `examples/`

### Step 2 ‚Äî Copy ts-port-app/ ‚Üí app/

```bash
rsync -av --exclude='node_modules' --exclude='dist' \
  helios-1/ts-port-app/ helios/app/
```

### Step 3 ‚Äî Copy ts-port-nestjs/ ‚Üí packages/nestjs/

```bash
mkdir -p helios/packages
rsync -av --exclude='node_modules' --exclude='dist' \
  helios-1/ts-port-nestjs/ helios/packages/nestjs/
```

### Step 4 ‚Äî Copy plans + .opencode

```bash
mkdir -p helios/plans helios/.opencode/plans
cp helios-1/plans/TYPESCRIPT_PORT_PLAN.md helios/plans/
cp helios-1/.opencode/plans/*.md helios/.opencode/plans/
```

### Step 5 ‚Äî Copy + adapt loop.sh

Copy `helios-1/loop.sh` ‚Üí `helios/loop.sh`

Adaptations needed:
- Remove `TS_PORT="$ROOT/ts-port"` ‚Äî the root IS the project now
- Remove all `JAVA_SRC_*` / `JAVA_TEST_*` variables (7 vars)
- In the prompt template: replace all `%%TS_PORT%%` with `%%ROOT%%`
- Remove all `Java source roots:` lines from prompt
- Remove all Java-related sed substitutions (7 lines in both Darwin and non-Darwin branches)
- Keep the core loop logic, PLAN path, and prompt structure

### Step 6 ‚Äî Fix path aliases

**`helios/app/tsconfig.json`** (line 15):
```
BEFORE: "@helios/*": ["../ts-port/src/*"]
AFTER:  "@helios/*": ["../src/*"]
```

**`helios/packages/nestjs/tsconfig.json`** (line 18):
```
BEFORE: "@helios/core/*": ["../ts-port/src/*"]
AFTER:  "@helios/core/*": ["../../src/*"]
```

### Step 7 ‚Äî Create .gitignore

```gitignore
node_modules/
dist/
.DS_Store
.idea/
*.tsbuildinfo
```

### Step 8 ‚Äî Install dependencies

```bash
cd helios && bun install
cd helios/app && bun install
# packages/nestjs uses workspace:* dep on @helios/core ‚Äî skip until workspace is set up
```

### Step 9 ‚Äî Run tests (verification)

```bash
cd helios && bun test            # expect: 2105 pass, 0 fail
cd helios && bun run tsc --noEmit  # expect: 0 errors
cd helios/app && bun test        # expect: 25 pass, 0 fail
```

### Step 10 ‚Äî git init + initial commit

```bash
cd helios
git init
git add -A
git commit -m "feat: extract Helios TypeScript port into standalone repository

Flattened ts-port/ to repo root. ts-port-app/ ‚Üí app/. ts-port-nestjs/ ‚Üí packages/nestjs/.
2105 core tests + 25 app tests passing."
```
