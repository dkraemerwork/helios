# Plan: Extract `@helios/nestjs` + Modern NestJS Library Patterns

> Full plan now lives in `plans/TYPESCRIPT_PORT_PLAN.md` Phase 9 (Blocks 9.0–9.9).
> This file contains the original extraction-only plan for reference.

## Goal

Extract the 14 NestJS source files and 11 test files from `ts-port/src/nestjs/` into a new standalone `ts-port-nestjs/` package (`@helios/nestjs`). Set up Bun workspaces. Rename core package to `@helios/core`. The core package must have zero NestJS dependencies.

## Key Decisions

- **Workspace**: Bun workspaces via root `package.json`
- **Location**: `ts-port-nestjs/` alongside `ts-port/`
- **Package names**: `@helios/core` (renamed from `helios`) + `@helios/nestjs` (new)
- **Target audience**: NestJS developers
